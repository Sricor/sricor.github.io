变量的作用域

作用域是一个项在程序中有效的范围。

{                      // s 在这里无效，它尚未声明
	let s = "hello";   // 从此处起，s 是有效的
    ...
}                      // 此作用域已结束，s 不再有效

变量 s 绑定到了一个字符串字面值，这个字符串值是硬编码进程序代码中的。

这里有两个重要的时间点：

  • 当 s 进入作用域 时，它就是有效的。
  • 这一直持续到它 离开作用域 为止。

引用的作用域

引用的作用域是从声明的地方开始一直持续到最后一次使用为止
